# 💱 Java Currency Converter

A simple Java-based application that converts currency values between different countries using real-time or predefined exchange rates.

---

## 📌 Overview

This project demonstrates core Java concepts by building a functional currency converter that performs accurate conversions based on exchange rates.

---

## ✨ Features

* 💱 Convert between multiple currencies
* ⚡ Fast and efficient calculations
* 🧮 User-friendly input/output
* 🔄 Supports dynamic conversion logic

---

## 🛠️ Tech Stack

* Java
* OOP (Object-Oriented Programming)
